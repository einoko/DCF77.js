DCF77.js

DCF77 Web Audio Simulator

A proof-of-concept DCF77 time signal simulator written in JavaScript, using Web Audio API. This page works correctly on Firefox and Chrome. You can use this web page to synchronise your DCF77-radio-controlled clocks/watches. Set your radio controlled clock/watch to “receive” mode and place it near the speaker or headphones. It should take 2–10 minutes for your watch to synchronize.

The transmitted signal is automatically in correct UTC+1 timezone (DCF77 signal time). Some filtering is applied to the signal, to make it less harsh on ears and to improve the signal quality. Nevertheless, the signal is a high-pitched 15.5kHz sine wave, so please protect your hearing! The signal has been successfully tested on a Casio Wave Ceptor watch and a radio controlled wall clock.

This project was inspired by these resources:
